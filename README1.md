# 📚 AI Smart Study Assistant

A web-based smart study assistant designed to help students manage tasks, track progress, and get instant help using an AI-style chatbot.

## 🚀 Features
- 📋 Task management with priority & deadline
- 📊 Progress tracking
- ⏱️ Pomodoro timer
- 🤖 AI-style chatbot (multi-domain queries)
- 🌙 Dark mode UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🌐 Live Project
https://ai-study-assistant-rajeshwari.netlify.app

## 📌 Description
This project simulates an AI-powered assistant using rule-based logic to handle queries across study, health, and general topics.
